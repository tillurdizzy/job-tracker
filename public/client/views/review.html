D. Baseline Proposal and Scope of Work

The following installation represents a baseline roof assembly, which meets or exceeds all applicable building codes. However, these are the minimum standards. Evolution Roofing highly recommends upgrading several of these components in the following Premium Upgrade Options section.

1. Pre-install

Remove existing roof system from the roof deck and haul from the site. Prepare the roof deck by nailing all raised nails and clearing debris from the roof surface left from the previous application.

2. Felt Underlayment

Install new #15 Asphalt Saturated Organic Felt underlayment over entire roof deck.

Learn More

3. Drip Edge Extender (if selected)

See below to select this Premium Upgrade.

4. New Drip Edge

Install 2" x 2" pre-painted to match the fascia or the roof color as chosen by owner (minimum IBC Building Code Standard).

Learn More

5. Valley Flashing

Install 36" wide valley underlayment covered with laced shingles; tail ends extending a minimum of 1" past the fascia board intersection.

Premium upgrade available below for this product.

Learn More

6. Starter Shingles

Install Pre-cut starter strips.

Learn More

7. Field Shingles

Install standard three-tab economy field shingles using four (4) nails per shingle. High wind or steep pitch application shall receive six (6) nails per shingle in accordance with manufacturers specifications when applicable.

Premium upgrade available below for this product.

Learn More

8. Attic Ventilation

Install ridge vent attic ventilation system in accordance with the manufacturer's recommendations.

Learn More

9. Ridge Shingles

Economy-grade shingles along hips and ridges and over the ventilation system.

Premium upgrade available below for this product.

10. Pipes and Vents

Install new plumbing pipes and vent stacks as needed, and paint to match roof color as close as possible

Evolution Roofing, Inc. is to be notified within 24 hours from the time that a leak is detected. Evolution Roofing, Inc. will not be held liable for consequential damages beyond the materials installed by Evolution Roofing, Inc. after completion of the roofing system. Settlement of the building foundation or repairs could void your warranty. Customers having foundation repairs shall notify Evolution Roofing, Inc. for an inspection of roofing system within 2 weeks of said repairs for continued roof warranty coverage. Any deviations to the building after completion without notifying Evolution Roofing, Inc. will void the water tightness warranty of the roof system if notice is not sent to Evolution Roofing to ensure water tightness. Evolution Roofing will provide repairs to the roofing system for a minimum trip charge of $125.00 for all non-warranty claims as a minimum, with repairs exceeding a (1) hour repair, if needed, to be billed at the owner’s expense to bring the building into warrantable condition.
