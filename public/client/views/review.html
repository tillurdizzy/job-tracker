Baseline Proposal and Scope of Work

Evolution Roofing Systems recommends the following as a baseline roof assembly. Options for a Premium Upgrades to follow.

1. Pre-install

Remove existing roof system from the roof deck and haul from the site. Prepare the roof deck by nailing all raised nails and clearing debris from the roof surface left from the previous application.

2. Install Felt Underlayment

Install new #15 Asphalt Saturated Organic Felt underlayment over entire roof deck.

3. Install Premium Trim (if selected)

See below to select this Premium Upgrade.

4. Install New Drip Edge

2" x 2" pre-painted to match the fascia or the roof color as chosen by owner (minimum IBC Building Code Standard).

5. Install New Valley Flashing

36" wide valley underlayment covered with laced shingles; tail ends extending a minimum of 1" past the fascia board intersection.

Premium upgrade available below for this product.

6. Install Starter Shingles

GAF Pro-Start Strip Shingles

Starter shingles are the first row of shingles installed along the entire perimeter of the roof. Evolution will install GAF Pre-cut Starter Strips. Some contractors try to use a cut-up field shingle as the 'starter' course on your roof. Unfortunately, cut-up shingles may not have properly positioned adhesive to ensure adequate wind resistance.

7. Install Field Shingles

GAF Timberline Natural Shadow

Evolution will install the above GAF Timberline Field shingles using four (4) nails per shingle. High wind or steep pitch application shall receive six (6) nails per shingle in accordance with manufacturers specifications when applicable.

Premium upgrade available below for this product.

8. Attic Ventilation

Cobra Ridge Vent 3

Evolution will install a ridge vent attic ventilation system in accordance with the manufacturer's recommendations.

9. Ridge Shingles

GAF Royal Sovereign

Evolution will install standard ridge shingles over the ventilation system.

Premium upgrade available below for this product.

10. Pipes and Vents

Evolution will install new plumbing pipes and vent stacks as needed, and paint to match roof color as close as possible
